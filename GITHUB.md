# VSTECH Informática

Catálogo online de produtos da VSTECH Informática.

## Publicação

O site está hospedado no GitHub Pages e pode ser acessado pelo link do projeto.

## Atualizações

Para atualizar o catálogo, substitua os arquivos necessários e publique novamente no GitHub.

## Imagens

As imagens dos produtos devem estar na pasta `imagens` e precisam estar com os nomes corretos para aparecerem no site.
